Inputs

Inputs fields can be used data collection. Feedback about data can be also be provided via Input Filed Styles. If a user hovers over input field or is interacting with , input field 1 can used to indicate focus. Input fields 2 & 3 can be used for error feedback. To disable any input field, add is-disabled to class of an input element.
