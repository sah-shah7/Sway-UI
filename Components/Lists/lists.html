Lists

Lists are used to present text or data in a structured format.

Ordered List

Ordered List includes arrangement of list with a number or an alphabet. The order can be ascending or descending. The pre-defined lists style in Sway UI are Numbers, captial alphabets, lowercase alphabets. To use list style, copy the corresponding class name and add it to li or ol element.
