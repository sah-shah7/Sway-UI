Buttons

Buttons are used to trigger an acton when clicked on it.

Primary & Link Buttons

For Sway UI, the default buttons are categorized based on the action they should trigger. The default width here is set to fit content. To use fix size, overide the width property. The pre-defined buttons are:- is-primary,is-success,is-danger,is-link. To use the buttons copy the whole button element with the desired action.

Buttons with icon

Buttons with icons can be used to specify an action visually, providing more information. Eg:- Add to Cart button with cart icon indicating that this button will add an item to the cart. Another eg is settings button with gear icon. The buttons are based on the above & icons can added with font awesome library.

Floating Buttons

Floating buttons are stacked top on the main body. They can include various quick actions. For eg:- An upward arrow floating button will send to the top of the page. Another eg:- A floating button containing some quick actions shortcuts. Indicated here with the button in purple.
