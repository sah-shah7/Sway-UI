Text Utilities

Text Utilities are various predefined styles that can be used to style,align and use them instantly

Headings

To use heading styles, just the add class name of the desired font size to any h tags.

h1. This is a very large header.

h2. This is a large header.

h3. This is a medium header.

h4. This is a moderate header.

h5. This is a small header.
h6. This is a tiny header.
