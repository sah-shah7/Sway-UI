# Sway-UI

Sway UI is a css component libray that is ready sway over your website.

## Description
 A component library built with html, css & js. It is user-friendly,flexible,fast,responsive & consistent.

## Instructions

To use Sway UI components in your next porject, just add the following line of code in the head section of your HTML file.

```

<link rel="stylesheet" href="https://sway-ui.netlify.app/main.css">

```
